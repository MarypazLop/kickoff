CÓMO EJECUTAR LA APLICACIÓN

1. Abra una terminal dentro de esta carpeta.
2. Ejecute:

   npm start

3. Abra en el navegador:

   http://localhost:5500

IMPORTANTE:
- Ya no use Live Server.
- Ya no abra index.html con doble clic.
- El servidor incluido funciona como intermediario para evitar el bloqueo CORS.
